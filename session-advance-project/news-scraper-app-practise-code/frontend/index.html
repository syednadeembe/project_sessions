About This News Scraper

Search Engine Used: We scrape Bing News results using BeautifulSoup (Python library).

Frontend: HTML, CSS, JavaScript frontend, styled like Google's homepage.

Backend: FastAPI server handles search, checks MongoDB cache first, scrapes if needed.

Database: MongoDB persists all search queries and scraped results for fast reuse.

Setup Workflow: The entire system runs via Docker Compose: frontend (Nginx), backend (FastAPI), database (MongoDB).

Learning Outcome: Teaches students full-stack concepts: frontend UI, API integration, scraping, caching, containerization.
